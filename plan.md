# План тестирования.
## Описание классов и методов для тестирования:

Описание основных классов и их методов
-Validator:
    -typeError - ошибка типа вводимых данных
    -valueError - ошибка значения вводимых данных
- Shape:
    - GetPerimeter - периметр фигуры
    - ShowInfo - вывести характеристики фигуры
    - GetArea - площадь фигуры
- Square(double length) - Квадрат, входные параметры: длина стороны: тип число
    - GetArea - площадь 
    - GetPerimeter - периметр
    - GetDiagonal - диагональ
    - ShowInfo - вывести характеристики фигуры
- Rectangle(double width, double height) - Прямоугольник. входные параметры: длина и ширина: тип число
    - GetPerimeter - периметр фигуры
    - ShowInfo - вывести характеристики фигуры
    - GetArea - площадь фигуры
    - GetDiagonal - диагональ
- Circle(radius: float) - Круг. входные параметры: радиус: тип число
    - GetPerimeter - периметр фигуры
    - ShowInfo - вывести характеристики фигуры
    - GetArea - площадь фигуры
    - GetDiameter - диаметр
- Triangle(side1: float, side2: float, side3: float) - Треугольник. входные параметры: сторона1, сторона2, сторона3: тип число
    - PrivateTriangleValidation - специальная проверка соотношения сторон прямоугольника 
    - GetPerimeter - периметр фигуры
    - ShowInfo - вывести характеристики фигуры
    - GetArea - площадь фигуры
    - GetHigh - высота


## CalcTest:
### Square
Цель: проверить вычисление радиуса окружности
    - 3 = > result > 4

### Rectangle
Цель: проверить правильность вычисления диагонали прямоугольника
    -  3, 4 => 5

## Проверка входных данных
### Triangle
Цель: проверить создание класса Triangle, и вычисление основных его характеристик
- Тесты расчёта площади
    - 1, 1, 1 => [0..100] 
    - 1, 1, 1 => !=-1
- Тесты расчёта периметра
    - 1, 1, 1 => [1..100]
    - 1, 1, 1 => != -1

### Тесты валидатора
    - 0, 0, -1, 0 => False
    - 0, -1, -1, 0 => False
    - 10, -1, -1, 0 => False
